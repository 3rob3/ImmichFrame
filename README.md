This project was created in VS Code using the Avalonia extension. It is a .NET 7 cross platform project and currently supports Linux, MacOS, and Windows. I am running it on a Raspberry Pi 5. You can learn more about Avalonia here:
https://avaloniaui.net/
This connects to a running Immich server using their API, and makes getRandom calls to randomly show images in your collection and fileCreatedAt date. Configure your server URL, account credentials, and interval in Settings.txt file.
